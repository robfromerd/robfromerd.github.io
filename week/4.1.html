Code Sample 4.1: Getting Started with Understrap and Woocommerce

Setting Up Understrap and Understrap Child

  1. Start and login to your Local by Flywheel Site (append /wp-admin)
  2. From the Dashboard, go to Appearance > themes > add new > search for Understrap
  3. Install Understrap
  4. Download the Understrap child theme from this Github repository (click green code button and then choose download Zip): https://github.com/understrap/understrap-child
  5. Unzip it if you need to and then you will have a folder called "understrap-child-master"
  6. Put this folder at the following location in your Local WordPress site: app > public > wp-content > themes (WordPress child themes sit alongside parent themes in the themes directory because the child theme has code in it that refers to the name of the parent)
  7. From Appearance themes, first activate Understrap, then Understrap-child
  8. Browse to the site to see if it has changed from the default theme you started with

Installing Woocomerce

  1. Install the Woocommerce plugin by going to Plugins > Add New
  2. Activate the Woocommerce plugin and go through the Setup Wizard -- Just make some guesses as to the correct choices (can change later)
  3. Add one product by going to Products > Add New
  4. Browse the site and notice that you lack any nav items.
  5. Go to Appearance > menus where you will create a new menu (call it top nav), choose the "Primary Menu" checkbox, and then save it. Once it has been saved, add the Shop and Cart pages to the menu with the Add to Menu choice and click Save Menu.
  6. Browse the site to the shop page and add the product you made to your cart.
  7. Browse the site to the cart page and and for next week do the exercise in the code sample below.
